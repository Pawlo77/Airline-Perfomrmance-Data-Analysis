Welcome to Airline Performance Data Analysis’s documentation!

Overview

Described here python code’s goal is to quickly load and preprocess data, than to perform some queries on it in order to anwser some interesting questions combined in presentation available on project’s repo.

Note

This project was developed as part of data science course at Warsaw University of Technology
